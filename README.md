# Block Puzzle - Genshin Impact Theme

A relaxing block puzzle game (Tetris-style drag & drop) with a visual theme inspired by **Genshin Impact**.

![Game Screenshot](https://raw.githubusercontent.com/placeholder/screenshot.png)

## üéÆ How to Play

1.  **Drag and Drop**: Drag the pieces from the bottom dock onto the 10x10 grid.
2.  **Fit the Blocks**: Try to fit the pieces to create full horizontal or vertical lines.
3.  **Clear Lines**: Completing a line clears it and awards points.
4.  **Game Over**: The game ends when there is no space left for the available piece.
